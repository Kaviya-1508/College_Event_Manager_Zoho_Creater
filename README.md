# Campus Event Manager App

## Developed Using
Zoho Creator (Low-code platform)

## Description
This application manages college events, registrations, attendance tracking, and analytics dashboards.

## Features
- Event creation
- Student registration
- Seat limit validation
- Attendance tracking
- Reports and dashboard analytics

## Modules
1. Events
2. Registrations
3. Reports
4. Dashboard

## Workflow Automation
- Prevent overbooking
- Auto event status update
- Reminder notifications
